Приветствуем на тестовом задании!

Выполните следующие задачи для выполнения задания:

  • Возьмите ваш готовый код из задания 2 (БЭМ) и совместите с готовым кодом из задания 3 (JS) так, чтобы всё работало.
  • Для всех слоев задайте рамки, border-radius, отступы, размеры шрифтов и еще какой-нибудь красоты
  • Установите Gulp и соберите простую сборку с тремя тасками: sass, js, build
  • Таск Sass должен компилировать sass файлы стилей (которые вы создали во втором шаге) в один сжатый файл app.min.css. Файлов должно быть по количеству блоков в понятии БЭМ и именоваться точно так же + один sass файл с переменными (задайте единый цвет рамок и единый отступ внутри всех блоков)
  • Таск JS склеивать библиотеку jQuery (скачайте и установите в сборку через npm) и ваш объект App и минифицировать всё это в файл app.min.js
  • Таск Build должен по очереди запускать таски sass и js
  • Сборка должна работать и выдавать рабочую HTML страницу с активным JS содержимым
