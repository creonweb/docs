# Конфиг для создания группы пользователь Manager с урезанными правами

+ Зайти в Контроль доступа -> Политики доступа -> Импорт
+ Загрузить XML файлик 
+ Создать группу пользователей Subadmins
+ Зайти в Группы пользователей & Пользователи -> Subadmins -> Редактировать группу пользователей
+ Настроить доступ к контекстам - поставить для mgr и web политику Subadmins
+ Добавить еще один доступ к контексту mgr политику miniShopManagerPolicy (если установлен MiniShop2)
